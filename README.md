## Featured Projects

### **1. Comparative Analysis of RAG-Based LLMs for Medical Chatbots**
Fine-tuned open-source LLMs (Flan-T5, LLaMA-2, Mistral-7B) using LoRA/PEFT techniques for medical Q&A. Developed a novel logit-based MCQ evaluation approach achieving **57% accuracy on USMLE** exams. Built a Flask & JavaScript medical chatbot with embedding models.
- **Tech Stack:** PyTorch, Hugging Face, LangChain, Flask, JavaScript
- [GitHub Repo](https://github.com/arunabh-alt/Comparative-Analysis-of-RAG-based-LLMs-for-Medical-Chatbot.git)

### **2. Image-Text Matching Multi-Modal Classifier**
Designed and trained a multi-modal deep learning classifier achieving **87% accuracy** on the Flickr8K dataset. Employed ResNet50V2 for vision encoding and BERT/GPT-2 for text embeddings with contrastive learning.
- **Tech Stack:** TensorFlow, Keras, Transformers, ResNet50
- [GitHub Repo](https://github.com/arunabh-alt/Image-Text-Match-Deep-Learning-Models.git)

### **3. Warehouse Robot Trajectory Optimization with RL**
Implemented Q-learning and SARSA algorithms with human-in-the-loop feedback for robot navigation in dynamic warehouse environments. Designed reward shaping strategies for efficient convergence in sparse reward settings.
- **Tech Stack:** PyTorch, OpenAI Gym, ROS2, Gazebo
- [arXiv Paper](https://arxiv.org/abs/2407.11671)

### **4. Super-Mario-Bros Deep Reinforcement Learning Agent**
Developed and compared RL agents (PPO, DQN, A2C) for dynamic gameplay optimization, achieving competitive performance through policy gradient methods and value-based learning.
- **Tech Stack:** PyTorch, Gym, Stable-Baselines3
- [GitHub Repo](https://github.com/arunabh-alt/Super_Mario_Bros_Deep_Reinforcement_Learning_Agent.git)
