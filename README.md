## Featured Projects

### Warehouse Robot Trajectory Optimisation with RL
Implemented Q-learning and SARSA with human-in-the-loop feedback for robot navigation in dynamic warehouse environments. Systematic benchmarking across exploration strategies and reward shaping methods.
- **Stack:** PyTorch · ROS2 · Gazebo · OpenAI Gym
- [arXiv:2407.11671](https://arxiv.org/abs/2407.11671)

### RAG-Based LLMs for Medical Chatbots
Fine-tuned Llama-2-7B, Mistral-7B, and Flan-T5 with LoRA/PEFT — 32% accuracy gain, 70% model size reduction. Introduced novel logit-based MCQ evaluation methodology.
- **Stack:** PyTorch · Hugging Face · LangChain · Flask
- [MDPI MAKE 2024](https://www.mdpi.com/2504-4990/6/4/116) · [GitHub](https://github.com/arunabh-alt/Comparative-Analysis-of-RAG-based-LLMs-for-Medical-Chatbot.git)

### Image-Text Multi-Modal Classifier
Novel multi-modal architecture achieving 87% accuracy on Flickr8K. ResNet50V2 vision encoder paired with BERT/GPT-2 text encoders using contrastive learning.
- **Stack:** TensorFlow · Keras · Transformers
- [GitHub](https://github.com/arunabh-alt/Image-Text-Match-Deep-Learning-Models.git)

### Super Mario Bros Deep RL Agent
Compared PPO, DQN, and A2C agents for dynamic gameplay optimisation using policy gradient and value-based methods.
- **Stack:** PyTorch · Stable-Baselines3 · Gym
- [GitHub](https://github.com/arunabh-alt/Super_Mario_Bros_Deep_Reinforcement_Learning_Agent.git)
